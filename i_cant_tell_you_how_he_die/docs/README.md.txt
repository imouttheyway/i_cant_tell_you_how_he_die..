👻 Programme Halloween - L'entité vous observe... 🎃

Ce projet n'est pas qu'un simple programme Python... C'est une expérience. Une immersion dans une atmosphère inquiétante, où l'illusion d'une entité numérique tente de vous perturber. Mais ne vous inquiétez pas... tout ceci n'est qu'un jeu. Ou du moins, c'est ce que vous voulez croire.

📌 Fonctionnalités

Messages envoyés par "l'entité", au hasard... ou pas.

Bruits étranges qui surgissent sans prévenir.

Attentes troublantes, comme si quelque chose observait.

Simulations d'erreurs système inquiétantes.

Une immersion totale dans un univers angoissant. 💀



🚀 Installation et Exécution

1️⃣ Prérequis

Python 3.x installé


3️⃣ Exécuter le script

python main.py

🎧 Astuces pour une expérience optimale

Lancez le programme dans une pièce sombre.

Mettez un casque... si vous osez entendre ce qu'il a à dire.

Rappelez-vous : ce n'est qu'un programme. Rien de plus... enfin, normalement. 👀
